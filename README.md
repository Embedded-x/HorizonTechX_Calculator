# Modern Calculator

A modern and responsive calculator built using **HTML, CSS, and JavaScript** as part of the **Horizon Tech X Internship**.

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ⌨️ Keyboard Support
* 🗑️ Clear (`C`) and Backspace (`⌫`) Functionality
* 📜 Calculation History
* 💾 Persistent History using `localStorage`
* 🗑️ Clear History Button
* 🌙 Dark/Light Theme Toggle
* 📋 Copy Result to Clipboard
* ⚠️ Error Handling for Invalid Expressions
* 🚫 Prevention of Invalid Inputs
* 📱 Fully Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
* Clipboard API
